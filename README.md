# Features
* Auto-save 
* Medias upload support
* HannaCode support

## Blocks Implemented :
* Heading
* Image (ProcessWire Native Support)
* Paragraph
* Embed
* Quote
* Code
* Link
* Table
* Delimiter
* Raw HTML
* Note (custom block markup)
